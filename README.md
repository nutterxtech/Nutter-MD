> A powerful WhatsApp multi-device bot by [@nutterxtech](https://github.com/nutterxtech)

## What is NUTTER-XMD?

NUTTER-XMD is a WhatsApp bot that runs on your own Heroku instance. It supports essential automation and is fully controlled via WhatsApp commands.
